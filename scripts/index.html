Аппаратная часть

🖥️

Raspberry Pi 5

Центральный вычислительный модуль системы, который управляет камерой и обрабатывает изображения археологических объектов в реальном времени.

📸

Камера USB

Используется для точной фотофиксации находок и позволяет различать мелкие детали: трещины, орнаменты и следы обработки.

🔋

Портативное питание

Обеспечивает автономную работу системы в полевых условиях без подключения к электросети.

🛡️

Защитный корпус

Защищает оборудование от пыли, влаги и механических повреждений при работе на раскопках.

🐍

Python + PyTorch

Основной язык программирования с фреймворком глубокого обучения для создания нейронных сетей.

👁️

OpenCV

Библиотека компьютерного зрения для анализа изображений, выделения контуров и распознавания объектов.
